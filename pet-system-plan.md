# 펫 시스템 기획안

## 📋 개요
동물의 숲 스타일 게임의 펫 시스템 기획안입니다. 플레이어가 다양한 동물을 키우고, 상호작용하며, 함께 탐험할 수 있는 시스템입니다.

---

## 🐾 1. 펫의 종류

### 1.1 기본 펫 카테고리
- **소형 동물**: 고양이, 강아지, 토끼, 햄스터, 새 (앵무새, 참새 등)
- **중형 동물**: 여우, 너구리, 다람쥐, 오리, 거북이
- **대형 동물**: 곰, 늑대, 사슴, 말, 양
- **특수 동물**: 펭귄, 팬더, 고릴라, 호랑이, 사자 (희귀)

### 1.2 펫 타입별 특징
- **따라다니는 타입**: 플레이어를 따라다니며 아이템을 찾아주거나 수집 도움
- **집 지키는 타입**: 집 근처에서 머물며 잡초 제거, 작물 보호
- **탐험 타입**: 멀리 떨어진 곳에서 희귀 아이템 발견
- **조수 타입**: 채집/농사/낚시 등 특정 활동 도움
- **장식 타입**: 단순히 함께 있는 것을 즐기는 펫

---

## 🎁 2. 펫 획득 방법

### 2.1 기본 획득 방법
1. **야생 동물과의 우정**
   - 야생 동물에게 매일 음식 주기
   - 특정 아이템 선물하기
   - 7일 연속 상호작용 시 우정 형성
   - 우정 형성 후 "함께 살자" 제안

2. **상점 구매**
   - 펫 상점에서 펫알/펫 사료 구매
   - 가격: 1000-5000 코인
   - 희귀 펫은 더 비쌈

3. **이벤트/미션 보상**
   - 계절 이벤트 참여
   - 특별 미션 완료
   - NPC 선물

4. **발견/구조**
   - 맵 곳곳에서 부상당한 동물 발견
   - 구조 후 치료하면 펫으로 영입
   - 숨겨진 장소에서 펫알 발견

5. **번식**
   - 같은 종류의 펫 2마리 이상 키우면 번식 가능
   - 특별한 날짜에 번식 이벤트 발생
   - 새끼 펫은 부모의 특징을 물려받음

## 🏠 3. 펫 관리 시스템

### 3.1 펫 집 (펜/하우스)
- **기본 펫집**: 펫 1마리 수용 (무료)
- **확장 펫집**: 펫 3마리까지 수용 (1000 코인)
- **대형 펫집**: 펫 5마리까지 수용 (5000 코인)
- **펫 하우스 업그레이드**: 
  - 실내/실외 선택
  - 장식 추가 (침대, 장난감, 급식기 등)
  - 색상/스타일 커스터마이징

### 3.2 펫 상태 관리
- **건강**: 0-100 (일정하지 않으면 감소)
- **행복도**: 0-100 (놀아주기, 음식 주기로 증가)
- **친밀도**: 0-100 (시간이 지날수록, 상호작용 시 증가)
- **배고픔**: 0-100 (음식 주면 증가, 시간 경과 시 감소)
- **에너지**: 0-100 (활동 시 감소, 휴식 시 증가)

### 3.3 펫 케어
- **음식 주기**: 펫 사료, 특별 음식 (친밀도 증가)
- **놀아주기**: 공 던지기, 애니메이션 상호작용
- **목욕시키기**: 깨끗하게 하면 행복도 증가
- **잠자리**: 펫집이나 특별 장소에서 휴식
- **산책**: 플레이어와 함께 맵 탐험

## 🎮 4. 펫 기능 및 능력

### 4.1 수집 도움
- **자동 수집**: 펫이 주변의 채집 가능한 아이템을 자동으로 수집
- **반경 수집**: 펫 주변 일정 반경 내 아이템 탐지
- **희귀 아이템 발견**: 특정 펫은 희귀 아이템을 찾는 능력 보유

### 4.2 전투/방어 (선택적)
- **야생 동물 위협**: 야생 동물이 접근할 때 펫이 경고
- **해충 방제**: 농장의 해충을 잡아먹음 (고양이, 새 등)

### 4.3 운반 능력
- **인벤토리 확장**: 펫이 추가 인벤토리 공간 제공
- **무거운 아이템 운반**: 특정 펫은 무거운 아이템을 운반 가능

### 4.4 탐험 능력
- **비밀 장소 발견**: 펫이 숨겨진 장소를 찾아냄
- **길 안내**: 목적지까지 펫이 길을 안내
- **밤 시야**: 어두운 곳에서 펫이 빛을 비춤

### 4.5 작업 도움
- **농장 작업**: 물주기, 수확 도움
- **건축 도움**: 자재 운반, 공구 전달
- **낚시 도움**: 미끼 가져오기, 물고기 운반

## 📈 5. 펫 성장 시스템

### 5.1 펫 레벨
- **레벨 1-10**: 새끼 펫
  - 기본 능력만 사용 가능
  - 성장 속도 빠름
  
- **레벨 11-30**: 성장 펫
  - 새로운 능력 습득
  - 외형 변화
  
- **레벨 31-50**: 성숙 펫
  - 모든 능력 사용 가능
  - 특별 스킬 보유

- **레벨 51+**: 완전 성숙 펫
  - 최고 능력치
  - 레어 스킬 발동

### 5.2 펫 진화/변화
- **외형 변화**: 레벨 업 시 외형이 변화
- **색상 변화**: 특정 조건 만족 시 색상 변형
- **크기 변화**: 성장에 따라 크기 변화
- **능력 변화**: 새로운 능력 습득

### 5.3 펫 특성
- **활발함**: 활동적인 펫 (높은 에너지)
- **조용함**: 차분한 펫 (집 지키기 좋음)
- **친화력**: 다른 펫과 잘 어울림
- **독립성**: 혼자서도 잘 지냄

## 💝 6. 펫과의 관계/친밀도

### 6.1 친밀도 단계
- **0-20**: 초면 (낯가림)
- **21-40**: 친구 (기본 상호작용)
- **41-60**: 절친 (특별 상호작용)
- **61-80**: 가족 (최대 능력 발휘)
- **81-100**: 영혼의 동반자 (특별 이벤트)

### 6.2 친밀도 증가 방법
- 매일 인사하기 (+1)
- 음식 주기 (+2)
- 놀아주기 (+3)
- 산책하기 (+2)
- 선물 주기 (+5)
- 특별 이벤트 참여 (+10)

### 6.3 친밀도에 따른 보상
- **높은 친밀도**: 더 많은 수집, 특별 아이템 선물, 펫의 특별 반응
- **낮은 친밀도**: 펫이 집을 떠날 수 있음 (경고 시스템)

## 🎨 7. 펫 커스터마이징

### 7.1 펫 액세서리
- **목걸이/인식표**: 펫 이름 표시
- **옷/의상**: 펫 전용 의상 (특별 이벤트)
- **리본/뱃지**: 장식 아이템
- **장난감**: 펫의 좋아하는 장난감

### 7.2 펫 집 장식
- **침대/쿠션**: 펫의 선호도에 따라 선택
- **장난감**: 다양한 장난감으로 놀아주기
- **급식기/물그릇**: 펫의 건강 관리
- **벽지/바닥**: 펫집 내부 장식

### 7.3 펫 이름 변경
- 펫에게 이름 지어주기
- 이름 변경 가능 (코인 소비)

## 🎯 8. 펫 관련 콘텐츠

### 8.1 펫 퀘스트
- **일일 퀘스트**: 펫에게 음식 주기, 산책하기
- **주간 퀘스트**: 펫과 함께 특정 장소 방문
- **특별 퀘스트**: 펫 관련 스토리 퀘스트

### 8.2 펫 이벤트
- **펫 생일**: 펫의 생일 파티 (선물 보상)
- **펫 대회**: 미용 대회, 능력 대회
- **펫 축제**: 특별 날짜에 펫 축제 열림

### 8.3 펫 상호작용
- **애니메이션**: 펫과의 다양한 애니메이션 상호작용
  - 쓰다듬기
  - 안아주기
  - 공 놀이
  - 간식 주기
- **대화**: 펫의 감정/상태를 텍스트로 표현
- **포즈**: 사진 모드에서 펫과 함께 포즈

## 📊 9. 펫 데이터 구조

### 9.1 펫 기본 정보
```typescript
interface Pet {
    id: string
    name: string
    type: 'cat' | 'dog' | 'rabbit' | 'fox' | ... // 펫 종류
    category: 'small' | 'medium' | 'large' | 'special'
    
    // 상태
    level: number
    experience: number
    health: number // 0-100
    happiness: number // 0-100
    intimacy: number // 0-100
    hunger: number // 0-100
    energy: number // 0-100
    
    // 능력
    abilities: PetAbility[]
    stats: {
        collectionRadius: number
        collectionSpeed: number
        inventorySlots: number
        specialSkill?: string
    }
    
    // 외형
    appearance: {
        color: string
        size: number
        accessories: string[]
    }
    
    // 메타 정보
    obtainedDate: number // 획득 날짜
    lastInteractionDate: number // 마지막 상호작용 날짜
    favoriteFood: string // 좋아하는 음식
    favoriteToy: string // 좋아하는 장난감
}
```

### 9.2 펫 능력
```typescript
interface PetAbility {
    id: string
    name: string
    description: string
    level: number // 필요한 펫 레벨
    cooldown: number // 쿨타임 (초)
    effect: {
        type: 'collect' | 'defend' | 'explore' | 'assist' | ...
        value: number
    }
}
```

- 음식 비용
- 치료 비용 (병에 걸렸을 때)
