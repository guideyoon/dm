
웹용 힐링 생활 시뮬레이션 게임 UI 전체 설계
기조 3D 플레이 화면을 중심으로 왼쪽 세로 메뉴와 오른쪽 컨텍스트 패널로 모든 기능을 처리한다
목표 3D 화면을 최대한 넓게 유지하고 클릭 2회 이내로 대부분의 행동에 도달한다
지원 PC 웹 모바일 PWA
모바일 세로 우선 설계, 데스크톱 경험을 해치지 않는다
